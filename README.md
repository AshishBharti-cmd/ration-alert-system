# Ration Alert System

## Overview

Ration Alert System is a cloud-based web application that helps ration card holders check the availability of wheat, sugar, and oil without visiting the ration depot unnecessarily.

In many local areas, people have to visit the depot multiple times or call other card holders to confirm whether ration has arrived. This causes confusion, wasted time, and inconvenience.

This project provides a simple and mobile-friendly solution where:

- Users can check real-time ration availability.
- Depot holders (Admin) can update ration status.
- Admin can add messages such as expected distribution date or reason for delay.
- Data is stored securely in Firebase cloud database.

---

## Features

- Live ration availability status
- Admin dashboard to update status
- Hindi interface for better accessibility
- Color-coded availability indicators (Green / Red)
- Cloud database using Firebase Firestore
- Mobile-first responsive design
- Hosted using Netlify

---

## Tech Stack

- HTML
- CSS (Mobile-first responsive design)
- JavaScript (ES Modules)
- Firebase Firestore (Cloud Database)
- Netlify (Deployment)

---

## How It Works

1. Admin updates ration availability from the Admin Dashboard.
2. Data is saved in Firebase Firestore.
3. Homepage fetches and displays updated status in real time.
4. Users can view latest availability and messages without visiting the depot.
